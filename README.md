# VITALIS Clínica Médica — Dashboard Financeiro

Projeto pronto para GitHub Pages.

## Como publicar

1. Abra o seu repositório no GitHub.
2. Faça upload de:
   - `index.html`
   - `vitalis-logo.png`
3. Vá em **Settings > Pages**.
4. Em **Branch**, selecione `main` e `/root`.
5. Aguarde alguns minutos e abra o link publicado.

## Ligações Google Sheets

O dashboard já está ligado aos CSV publicados das abas:
Saldos, Devedores, Despesas, Vendas, Financiamento, ERDM e Últimas Transações.

## Observação

O campo **Valor do Piso** é calculado automaticamente como:

`Total a Receber / 2`
